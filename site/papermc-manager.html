PaperMC Manager

PaperMC Manager is a CLI tool for the easy administration of Minecraft servers running the PaperMC custom server jar. The tool provices the ability to easily update the existing installation, or to fetch the needed files to begin a new installation.

The program is built in Python 3.8, leveraging Pyinstaller to build and distribute an "all-in-one" executable. It uses Click for the top-level CLI code, with Requests handling communication with the PaperMC api servers for downloading the server files. The program's code is tested and quality-checked using GitlabCI on my personal CI runners running in the Oracle Cloud. The runners also build the Pyinstaller executable and publish it as an artifact on the project page.
