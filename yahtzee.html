Yahtzee

This is a simulation of the dice game Yahtzee built in Java. The program utilizes Model-View-Controller design in order to accurately implement the game logic. The game is played via a GUI, which allows the user to roll and hold their dice, as well as choose how they want to score their dice at the end of their turn. The GUI uses Swing, and commands the model classes in order to change the game's state. I wrote this program as my end-of-semester group project for CSC116, and I have since updated it to use a proper package structure, as well as rewritten references to the dice image files to use Java URLs in order to make them display properly when the project is packaged into a JAR.
